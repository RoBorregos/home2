Detailed Description

Calculate a set of grasp candidates.

This class calculate sets of grasp candidates. The grasp candidates in the same set share the same point neighborhood and the same local reference frame (LRF). The grasps are calculated by a local search over discretized orientations and positions.

Note
For details, check out: Andreas ten Pas, Marcus Gualtieri, Kate Saenko, and Robert Platt. Grasp Pose Detection in Point Clouds. The International Journal of Robotics Research, Vol 36, Issue 13-14, pp. 1455 - 1473. October 2017. https://arxiv.org/abs/1706.09911
