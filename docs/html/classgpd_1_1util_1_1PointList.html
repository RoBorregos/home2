Detailed Description

List of points.

Stores a list of n points (3 x n matrix) together with their surface normals (3 x n matrix). Also keeps information about which camera sees which point by storing the view points (3 x k matrix), i.e., locations, of k cameras, and for each of the n points, if the point is seen or not from a camera (k x n matrix). If point i is seen from camera j, then cam_source(i,j) = 1. Otherwise, cam_source(i,j) = 0.
