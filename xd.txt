En la Orin, de cero:                                                                                                                                     
                                                                                                                                                           
  1. Traer el branch actualizado                                                                                                                           
  cd ~/home2          # o donde tengas el repo                                                                                                             
  git fetch origin                                                                                                                                       
  git checkout hri-ollama-cpp                                                                                                                              
  git pull                   
                                                                                                                                                           
  2. Descargar los modelos (solo la primera vez, ~5 GB para qwen3)                                                                                         
  cd docker/hri                                                                                                                                            
  bash scripts/download-model.sh                                                                                                                           
  # Selecciona: 1 (qwen3) y 2 (rbrgs)                                                                                                                      
  # Si quieres todo junto: escribe "1 2" o "a"                                                                                                             
  
  3. Build y arrancar (primera vez)                                                                                                                        
  ./run.sh --hric l4t --build                                                                                                                            
                                                                                                                                                           
  4. Verificar que los servidores levantaron (en otra terminal)                                                                                          
  curl http://localhost:11434/health   # qwen3                                                                                                             
  curl http://localhost:11435/health   # rbrgs (solo en gpsr)
                                                                                                                                                           
  5. Correr el benchmark                                                                                                                                   
  bash scripts/benchmark-llm.sh \                                                                                                                          
    "http://localhost:11434/v1" "llama.cpp-qwen3"                                                                                                          
                                                                                                                                                           
  ---                                                                                                                                                      
  Arranques siguientes (imagen ya construida):                                                                                                             
  cd docker/hri                                                                                                                                            
  ./run.sh --hric l4t                                                                                                                                    
                                                                                                                                                           
  Si quieres correr gpsr (levanta ambos servidores):                                                                                                       
  ./run.sh --gpsr l4t
